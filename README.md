# 🤖 Real-Time Hand Gesture Recognition

A real-time hand gesture recognition system built using **Python**, **OpenCV**, and **MediaPipe**. The application detects a user's hand through a webcam, tracks 21 hand landmarks, and recognizes common hand gestures in real time.

---

## 🚀 Features

- 📷 Real-time webcam detection
- 🖐️ Hand landmark tracking using MediaPipe
- ✊ Recognizes multiple gestures:
  - Open Palm
  - Fist
  - One Finger
  - Peace Sign
  - Thumbs Up
- ⚡ Fast and lightweight
- 💻 Easy to run

---

## 🛠️ Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/abhinavashish1710/Real-Time-Hand-Gesture-Recognition.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python hand_gesture.py
```
